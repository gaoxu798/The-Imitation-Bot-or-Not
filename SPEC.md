# The Imitation (Bot or Not) - MVP 需求文档

---

## 一、项目概述

**项目名称**：The Imitation（别名：Bot or Not）

**项目类型**：网页端竞技社交游戏

**核心理念**：玩家随机匹配AI或真人，随机分配"人类"或"AI"身份。作为"人类"需要在3次提问中猜出对方是AI还是人类；作为"AI"需要伪装自己不被识破。

**目标用户**：全球互联网用户，AI爱好者、社交游戏玩家、科幻迷

**slogan**：Can you tell bot from human?

---

## 二、核心玩法

### 2.1 匹配机制

- 随机匹配：玩家一键开始匹配，系统随机分配对手
- 对手类型：50%概率匹配AI，50%概率匹配真人玩家
- 身份分配：匹配成功后，随机分配"人类"或"AI"身份

### 2.2 游戏流程

1. 匹配阶段：点击"开始匹配"，等待系统配对
2. 身份揭示：双方同时看到自己的身份，但不知道对方身份
3. 问答阶段：
   - "人类"玩家拥有3次提问机会
   - 每次提问后，"AI/真人"玩家需在2分钟内回答
4. 猜测阶段：猜AI / 猜人类 / 不确定
5. 结果公布：双方看到匹配结果和得分

### 2.3 积分规则

| 结果   | 积分   |
| ------ | ------ |
| 猜对   | +10分  |
| 猜错   | -5分   |
| 不确定 | 0分    |

## 六、视觉设计（赛博朋克风格）

### 6.1 配色方案

| 元素       | 颜色代码 |
| ---------- | -------- |
| 主色-霓虹蓝 | #00F0FF |
| 主色-霓虹粉 | #FF00FF |
| 背景-深黑  | #0A0A0F  |
| 背景-次黑  | #151520  |
| 文字-白    | #FFFFFF  |
| 文字-灰    | #888888  |
| 强调-霓虹紫 | #9D00FF |
| 成功-霓虹绿 | #00FF88  |
| 错误-霓虹红 | #FF3366  |

### 6.2 背景效果

- 深色渐变背景 (#0A0A0F → #151520)
- 网格线叠加（半透明）
- 微妙的光晕/光束效果
- 局部使用故障(Glitch)效果

### 6.3 组件风格

- 按钮：玻璃拟态 + 霓虹边框 + 发光效果
- 卡片：半透明黑底 + 霓虹边框 + 模糊效果
- 输入框：Terminal终端风格 + 扫描线
- 图标：发光线条风格

### 6.4 字体

- 标题/Logo：JetBrains Mono / Fira Code（等宽科技感）
- 正文：Inter / Roboto（清晰易读）

### 6.5 关键页面视觉

**首页**
- 巨大霓虹Logo（带发光呼吸效果）
- "开始匹配"按钮（脉冲动画）
- 段位/胜率/排名用发光卡片展示

**游戏页**
- 问答框像Terminal终端
- 对方回答带CRT扫描线效果
- 倒计时用数字发光风格

**结果页**
- 猜对：绿色霓虹闪光 + "CORRECT"特效
- 猜错：红色霓虹闪烁 + "WRONG"特效

**个人中心**
- 数据用全息投影风格展示
- 体力条用能量条效果
